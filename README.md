## 🗺️ 60-Day Curriculum Roadmap

This challenge was structured into themed weeks to build a progressive mastery of Data Structures and Algorithms, moving from fundamental logic to complex system-level optimizations.

### 🟢 Phase 1: Foundations (Weeks 1–2)
* **Focus**: Data manipulation and basic search patterns.
* **Topics**: Arrays, Strings, Sorting, Prefix Sums, and Hashing.
* **Goal**: Mastering $O(N)$ lookups and cumulative data processing.

### 🟡 Phase 2: Linear Optimization (Week 3)
* **Focus**: Efficient subarray and subsegment processing.
* **Topics**: Sliding Window, Two Pointers, and Monotonic Stacks.
* **Goal**: Reducing $O(N^2)$ brute-force solutions to $O(N)$ linear scans.

### 🔵 Phase 3: Hierarchical Structures (Week 4)
* **Focus**: Tree traversals and recursive logic.
* **Topics**: Binary Trees, BST (Binary Search Trees), BFS/DFS, and Catalan Numbers.
* **Goal**: Understanding hierarchical data and optimal search properties.

### 🟣 Phase 4: Network & Connectivity (Week 5)
* **Focus**: Complex relationships and graph theory.
* **Topics**: Graphs (BFS/DFS), Dijkstra’s (Shortest Path), Tarjan’s (Articulation Points), Kruskal’s (MST), and Topological Sort.
* **Goal**: Implementing industry-standard graph algorithms for connectivity and routing.

### 🟠 Phase 5: Optimization Paradigms (Week 6)
* **Focus**: Decision making and bit-level efficiency.
* **Topics**: Dynamic Programming (DP), Greedy Algorithms, Bit Manipulation, and Matrix Traversal.
* **Goal**: Mastering state-based problem solving and memory-efficient logic.

### 🔴 Phase 6: Mastery & Integration (Weeks 7–8)
* **Focus**: Advanced problem solving and system-level parsing.
* **Topics**: String Parsing (Atoi), Mathematical Logic, Advanced 3D DP, and Complex Palindrome Logic.
* **Goal**: Synthesizing multiple patterns to solve "Hard" level competitive programming challenges.
